题目是通过设置代理，将网络请求发送到代理服务器，然后由代理服务器响应。

但是我按照文档给出的http://localhost:8888/www.google.com 的形式，将该形式的url发送给了代理服务器，然后代理服务器从本地文件将缓存信息返回，没有就由代理服务器发送给目的服务器http请求，再缓存http响应，再将缓存信息发送给浏览器
